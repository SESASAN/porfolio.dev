# Sebastián Pérez – Portfolio.dev

Este es mi portafolio web personal donde presento mis proyectos, habilidades y experiencia como desarrollador.  
Está construido con tecnologías modernas enfocadas en rendimiento, diseño limpio y buenas prácticas.

## Tecnologías utilizadas

- Astro
- Tailwind CSS
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## Estructura del proyecto

```
/
├── public/          # Recursos estáticos
├── src/
│   ├── components/  # Componentes reutilizables
│   ├── layouts/     # Layouts principales
│   └── pages/       # Páginas del sitio
├── package.json
└── astro.config.mjs
```

## Secciones del portafolio

- Inicio – Presentación breve
- Sobre mí – Información profesional
- Proyectos – Trabajos destacados
- Contacto – Formas de comunicación

## Instalación y uso

Clona el repositorio:

```bash
git clone https://github.com/SESASAN/porfolio.dev.git
```

Entra al directorio:

```bash
cd porfolio.dev
```

Instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```

El proyecto estará disponible en:

```
http://localhost:4321
```

## Build para producción

```bash
npm run build
```

Previsualizar el build:

```bash
npm run preview
```
